Перечень функций (все команды имеют префикс* "."):
1. fight - это функция для автоматизации боя, она принимает один аргумент в виде числа, это число является временем(в часах) на протяжении которого будет длиться авто-бой.
2. math - это функция используется для математических вычислений.
    1. Вы можете написать любой пример используя следующие операции:
        1. "+" - сложение
        2. "-" - вычитание
        3. "*" - умножение
        4. "/" - деление
        5. "//" - целочисленое деление
        6. "**" - возведение в степень
        7. "%" - получение остаткка от деления
    2. Так же вы можете использовоть скобки и числа с плавающей точкой(0.1)
3. sqrt - эта функция для извлечения квадратного корня из числа. Принимает один аргумент - это число без плавающей точки
4. hack - это развлекательная функция, она не принимает аргументов. Она выводит сообщение о взломе и прогресс в виде процентов, после завершения сообщается об успешном взломе после начинается поиск данных о существовании динозавров с процентом. После он выводит сообщение о находке данных о динозаврах.
5. type - это тоже развлекательная функция, она придает сообщению эффект печати.
6. case - это функция для открытия большого количества кейсов. Она принимает два аргумента, первый - это номер кейса, второй - это кол-во кейсов которое надо открыть. Она ещё отправляет 2 сообщения-отчета в избранное, первое - это сообщение о начале, в котором написано о кол-ве сообщений, время начала, примерное время выполнения и во сколько примерно закончится, второе - это сообщение об окончании. Там написано: 1 - кол-во отправленых сообщений и время отправки.
7. cw - эта функция не принимает аргументов, она покупает бафы для питомцев, начинает кв и 12 часов играет в авто-бой.
8. bonus - эта функция предназначена для игроков с вип статусом. Она не принимает аргументов
9. shrug - это функция изменяет сообщение на ¯\_(ツ)_/¯ и не принимает аргументов.
10. donate - эта функция предназначена для отправки игровой валюты игроку на протяжении нескольких дней. Она принимает следующие аргументы:
    1. id - id игрока
    2. сумма - валюта, которую вы хотите перевести
    3. время - время в днях
11. test - эта функция для проверки работоспособности программы, она изменяет сообщение на "Тест произведён успешно!"
12. info - информация о боте
13. city - это команда используется для строительста зданий в вашем городе. Она принимает следующие оргументы:
    1. тип построек(дом, вода, работа и т.д.)
    2. номер здания
    3. количество зданий
14. commands - выводит список команд с кратким описанием
15. db - эта команда для базы данных и использует следующие флаги:
    1. -a - добавить id канала в базу данных(принимает id канала в виде числа)
    2. -d - удалить канал(id канала)
    3. -v - посмотреть все записи
16. msg - отправка сообщений в чат
17. translate(tr) - переводчик
18. languege(lg) - выводит сообщение с доступными языкам
19. converter(con) - конверетер в валют
20. currencies(cur) - доступные валюты для конвертации

Префикс "." - то есть программа ответит на команду test если в её начале будет стоять "." (.test)
